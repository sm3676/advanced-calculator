# Advanced Calculator Application

## Overview
This project is an advanced calculator application built in Python using clean architecture and design patterns. The application supports arithmetic operations, logging, undo/redo functionality, configuration management, and persistent history storage.

## Features
- Basic arithmetic operations (Add, Subtract, Multiply, Divide)
- Factory Design Pattern
- Memento Pattern (Undo/Redo)
- Observer Pattern
- Logging to file
- CSV History storage
- Configuration via .env
- Input validation
- Custom exceptions
- 100% test coverage

## Design Patterns Used
- Factory Pattern
- Observer Pattern
- Memento Pattern
